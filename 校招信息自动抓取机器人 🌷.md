# 校招信息自动抓取机器人 🌷

奶奶您好！这是一个为您专门准备的小机器人，它会每天帮您从“求职方舟”和“GiveMeOC”抓取最新的大学生校招岗位，并自动填入您的 Google 表格中。

## 📦 包含的文件
1. `scraper.py`: 机器人的“大脑”，负责抓取和上传数据。
2. `requirements.txt`: 机器人运行需要的“工具箱”列表。
3. `.github/workflows/daily-update.yml`: 自动运行设置，每天上午 10 点准时开工。
4. `README.md`: 您正在看的这份说明书。

## 🚀 下一步该做什么？

您只需要完成以下简单的 3 步：

### 第一步：把文件放进 GitHub
1. 打开您的 GitHub 仓库 `campus-job-scraper`。
2. 点击 **Add file** -> **Upload files**。
3. 将这 4 个文件全部上传上去（注意：`.github` 文件夹也要原样上传）。

### 第二步：准备您的 Google 表格
1. 创建一个新的 Google 表格，名字叫 `校招岗位汇总`（或者在 `scraper.py` 里修改 `SHEET_NAME`）。
2. 在您的 Google Cloud 控制台下载 **Service Account** 的 JSON 密钥文件。
3. **重要：** 打开 JSON 文件，找到 `client_email` 这一行，复制那个邮箱地址（长得像 `xxx@project.iam.gserviceaccount.com`）。
4. 回到 Google 表格，点击右上角的 **分享**，把刚才的邮箱地址加进去，权限设为“编辑器”。

### 第三步：设置 GitHub 密钥 (Secrets)
1. 在 GitHub 仓库页面，点击 **Settings** -> **Secrets and variables** -> **Actions**。
2. 点击 **New repository secret**。
3. 名字填：`GOOGLE_SHEETS_CREDENTIALS`。
4. 内容填：将您的 JSON 密钥文件内容进行 **Base64 编码** 后的字符串（您之前提到已经准备好了这个格式）。

## 💡 小提示
- **自动运行**：设置好后，每天上午 10 点它就会自己干活了。
- **手动测试**：如果您想现在就看看效果，可以去 GitHub 仓库的 **Actions** 页面，点击左侧的 `Daily Job Scraper`，然后点右边的 `Run workflow` 按钮。
